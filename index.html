นโยบายความเป็นส่วนตัว (Privacy Policy)

สำหรับบริการระบบติดตามสถานะสินค้า ของ Smart Factory Solutions Co., Ltd. บน LINE LIFF

นโยบายความเป็นส่วนตัวนี้จัดทำขึ้นเพื่อชี้แจงรายละเอียดการเก็บรวบรวม ใช้ หรือเปิดเผยข้อมูลส่วนบุคคลของคุณ ในการเข้าใช้งานผ่านระบบ LINE LIFF ของเรา

1. ข้อมูลที่เราเก็บรวบรวม

เมื่อคุณใช้งานผ่าน LINE LIFF เราอาจเก็บข้อมูลดังต่อไปนี้:

  • ข้อมูลจาก LINE Profile: ได้แก่ ชื่อผู้ใช้งาน (Display Name), รูปโปรไฟล์ (Picture URL), สถานะ (Status Message) และ User ID (UID)
  • ข้อมูลที่คุณให้เพิ่มเติม: เช่น ระบุข้อมูลที่ขอ เช่น ชื่อ-นามสกุล, เบอร์โทรศัพท์, ที่อยู่, อีเมล
  • ข้อมูลทางเทคนิค: เช่น ข้อมูลอุปกรณ์, ระบบปฏิบัติการ และประวัติการใช้งานภายใน LIFF

2. วัตถุประสงค์ในการประมวลผลข้อมูล

เราจะนำข้อมูลของคุณไปใช้เพื่อวัตถุประสงค์ดังนี้:

  • เพื่อ ยืนยันตัวตนในการเข้าใช้งานระบบ
  • เพื่อ ให้บริการรายงานสถานะการสั่งซื้อสินค้า และเข้าร่วมเป็นสมาชิกในชุมชนของเรา
  • เพื่อ การสื่อสาร ให้ข้อมูลข่าวสาร หรือแจ้งเตือนสถานะต่างๆ ผ่าน LINE
  • เพื่อ พัฒนาและปรับปรุงประสิทธิภาพการขาย และส่งสินค้าได้อย่างโปร่งใส

3. การรักษาความปลอดภัย

เรามีมาตรการรักษาความปลอดภัยที่เหมาะสมเพื่อป้องกันไม่ให้ข้อมูลส่วนบุคคลของคุณสูญหาย เข้าถึงโดยมิชอบ หรือถูกเปิดเผยโดยไม่ได้รับอนุญาต โดยเราจะเก็บรักษาข้อมูลไว้เป็นระยะเวลา 1 ปี และ หรือ ตลอดระยะเวลาที่เป็นสมาชิก

4. สิทธิของเจ้าของข้อมูลส่วนบุคคล

ตามกฎหมาย PDPA คุณมีสิทธิในการ:

  • ขอเข้าถึงและขอรับสำเนาข้อมูลส่วนบุคคล
  • ขอให้แก้ไขข้อมูลให้ถูกต้องและเป็นปัจจุบัน
  • ขอให้ลบ หรือระงับการใช้ข้อมูลส่วนบุคคล
  • ถอนความยินยอมในการเก็บรวบรวมข้อมูลได้ทุกเมื่อ (ผ่านการตั้งค่าใน LINE หรือติดต่อเราโดยตรง)

5. การติดต่อเรา
